## Amazon Elasticache

- 완전관리형 in memory cache 서비스
→ 인메모리 캐시에서 가장 대표적인 redis와 memcached가 있다.
→ 인메모리 캐시 디비는 스토리지에 직접 접근하지않고 메모리로 모든 처리를 하므로 데이터의 저장이나 검색속도가 매우 빠름→  데이터는 메모리 크기까지만 저장 가능
→ 서버의 전원 공급이 중단되면 데이터 소멸
→ cluster mode 구축 가능
