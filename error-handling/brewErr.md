# Mac에서 brew install 오류문제 발생했을 때

**✓** d2 관련 구현하다가 curl. -fsSL https://~ 라는 명령어를 실행했는데도 관련 라이브러리가 설치되지 않았다

```yaml
설치 후 which -a d2라고 cli에 입력하면 찾을 수 없다고 계속 뜬다.
```

**✓** 그래서 문제가 되는 부분이라 생각한 `/opt/homebrew/bin/~`를 삭제했는데 아무래도 이부분이 문제가 됐는지,

**✓** 그 이후에 그 어떤 라이브러리도 brew로 설치하면 설치가 안됐다.

**✓** 리서치 결과 내가 지워버린 부분때문에 환경변수 경로에 문제가 생겼고

**✓** 가장 빠르게 해결할 수 있는 방법은 brew를 재설치하는 것이었다

**✓** 그래서 삭제하고 재설치했더니 제대로 작동했다.
