# EC2 재빌드 관련 에러

## ⚠️ issue

### ‼️ **문제 상황**
> → 현재 배포한 상태에서 key 페이지 데이터 안불러와짐. .env 문제인줄 알고 파일 업데이트했는데도 안됨. 내일 확인 예정 (에러확인결과 AWS Credential을 제대로 인식하지 못함 → 서버쪽 env 확인해보기)
> 
### **💡 해결 완료**
> → 문제는 **빌드** 때문이었다. 결과적으로는 env파일을 업데이트하지 않아서가 맞았던 것. 원래 env는 고쳐도 빌드를 다시 안해도 되는 줄 알았는데 그게 아니었다. 빌드를 다시 해주니 정상적으로 key 페이지의 데이터가 불러와졌다.
>
